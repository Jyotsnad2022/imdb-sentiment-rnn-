# IMDB Sentiment Classification — SimpleRNN

**Summary**  
Binary sentiment classification on the IMDB dataset using a SimpleRNN architecture built in Keras.

**Files**
- `IMDB_SimpleRNN.ipynb` — Colab notebook
- `requirements.txt`

**How to run**
1. Open the notebook in Colab and run all cells (Keras will download the IMDB dataset).
2. Observe training/validation curves and final evaluation.
